## ✨ Features

| Feature | Details |
|---|---|
| 🖱️ **Drag & Drop** | Reorder tasks by dragging — order persists to localStorage |
| 🗂️ **Categories** | Work, Study, Personal, Health, Finance, Other — color-coded |
| 📅 **Due Dates** | Due date labels: Today, Tomorrow, X days overdue |
| ⏰ **Reminders** | Auto-toast on load for overdue tasks + 1-day-before reminder |
| 💾 **LocalStorage** | All tasks, order, and theme persist across sessions |
| 🌙 **Dark / Light Mode** | Toggle with one click — preference saved |
| 🔍 **Search** | Live search filters tasks as you type |
| 🔴 **Priority Levels** | High / Medium / Low with color-coded left border |
| 📊 **Stats Bar** | Live count of Total, Active, Done, and Overdue tasks |
| 📈 **Progress Bar** | Animated completion percentage |
| ✏️ **Edit Modal** | Edit any task's title, category, priority, or due date |
| ⬇️ **Export** | Download all tasks as a `.json` file |
| 🗑️ **Clear Options** | Clear completed or clear all tasks |



## 🛠️ Tech Stack

- **HTML5** — Semantic structure, drag-and-drop API
- **CSS3** — CSS Variables, animations, responsive grid, dark/light theming
- **Vanilla JavaScript (ES6+)** — DOM manipulation, localStorage, event delegation, closures
- **No libraries. No build tools. No npm.**


## 💡 Key Concepts Demonstrated

- **DOM Mastery** — dynamic element creation, event delegation, classList manipulation
- **State Management** — single source of truth (`tasks[]` array), re-render on every state change
- **Drag & Drop API** — `dragstart`, `dragover`, `drop`, `dragend` events with index-based reordering
- **LocalStorage** — `JSON.stringify/parse` for persistence, theme preference saving
- **CSS Variables** — instant theme switching by toggling a single `data-theme` attribute
- **Event Delegation** — single listener on the list container instead of per-card listeners
- **Closures** — task ID captured in drag handlers
- **Responsive Design** — CSS Grid, flexbox wrapping, mobile breakpoints
